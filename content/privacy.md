This privacy policy covers this website (miniatures.saltares.com) and the publishing tools used to share its content on social media.

## This website

This site is a static blog. It does not collect, store, or process any personal data from visitors. There are no accounts, comments, contact forms, cookies, or analytics.

The site is hosted on GitHub Pages. GitHub may log standard technical information (such as IP addresses) for security and operational purposes — see the [GitHub Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement) for details.

## Social media publishing

The associated Meta app is a private, single-user tool used solely by the site owner to publish this blog's own content (images and captions) to the site owner's own Instagram account via the Instagram API.

This tool:

- does not collect, store, or share any data about any third party;
- does not interact with other users' accounts, content, or data;
- only sends the site's own publicly available content to Instagram.

## Data deletion

No user data is collected or stored, so there is nothing to delete. If you have any questions, contact the site owner at david@saltares.com.
